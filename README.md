# Portifólio Venâncio

## Como testar?

``` terminal
npm run dev
```

## Como exportar os arquivos estáticos?

``` terminal
npm run build
```
Gera na pasta `/out`.

## Como iniciou?

``` terminal
npx create-next-app
```

## Como servir a build estática?
``` terminal
npm install -g serve
```
Isso instala serve

``` terminal
serve out
```
isso serve a pasta out.
